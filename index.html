Pytest+ Jenkins + Allure Auto Framework Building

Prepare

Windows 10

Software/Plugins

python3
jenkins
java
allure

Steps

Python install

  1. download exe from https://www.python.org/downloads/release/python-370/
  2. install
  3. environment configuration setting:
  4. Compter> System>Advance System Settings>Environment>Path
  5. Check python installed or not， input python

Java install

  1. download package from https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
  2. install
  3. environment configuration setting:
  4. same as java
  5. Check java installed or not， input java -version in cmd

Tomcat installation

  1. Download tomcat
  2. https://tomcat.apache.org/download-90.cgi

Jenkins installation

  1. download : https://jenkins.io/download/
  2. http://localhost:8080/
  3. install the plungins which are recommanded, and set administors
  4. login

Allure installation

Trouble shooting

  1. Cannot run the software mainly due to the paths are not correct
  2. Jenkins can not run always due to the port is occupied
  3. If you forget you‘re jenkins’ administors password, you can do the following steps

One situation: password isn’t changed
A}Enter in ..\Jenkins\secrets, open initialAdminPassword file, copy password
B}Visit Jenkins webpage, enter admin and the password
C}Login and change the password

Second situation: password is changed
A)Delete config.xml under Jenkins file and save
