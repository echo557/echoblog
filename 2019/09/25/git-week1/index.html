git_week1

  1. 如何初始化
    git init
    git remote add origin https://github.com/echo557/git_study1.git
    git add .
    git commit -m comment
    git push –set-upstream origin master
    git branch –set-upstream-to=origin/ master

报错：
$ git pull
fatal: refusing to merge unrelated histories
分析：
他们是两个不同的项目，要把两个不同的项目合并，git需要添加一句代码.
解决：
$ git pull origin master –allow-unrelated-histories

  1. git checkout branch 创建branch
    –branch 是一个lable
    –A branch is a set of commits that trace back to the project’s first commit
    –Checkout involves updating HEAD and updating the working tree
    –A detached HEAD reference points directly to a commit
    –Fix a detached HEAD by creating a branch

git branch -b branchname ： HEAD 指针指向branchname
git branch -d branchname： Delete branch

  1. git reflog： return a local list
    git log –oneline –graph –all
  2. merging

–Fast-forward merge : Moves the base branch label to the tip of the topic branch
Condition: must in one line
Steps:
1)checkout master
2)Merge featureX
3)Delete the featureX branch

–Merge commit
–Squash merge
