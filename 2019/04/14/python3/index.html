from selenium import webdriver
from selenium.webdriver.common.action_chains import ActionChains
dr=webdriver.Chrome()
dr.get(“http://www.baidu.com")

right_click = dr.find_element_by_link_text(u”新闻”)

#右击
ActionChains(dr).context_click(right_click).perform()

#双击
ActionChains(dr).double_click(right_click).perform()

#拖拽
ActionChains(dr).drag_and_drop(element,target).perform()
element: 源位置
target: 目标位置

操作cookies

get_cookies() 获取所有cookies 信息
get_cookies(name) 返回有特定name值有cookies信息
add_cookies(cookie_dict) 添加cookies,必须有name和value值
delete_cookies(name) 删掉特定（部分）的cookie 信息
delete_all_cookies() 删除所有cookie信息

from selenium import webdriver
dr=webdriver.Chrome()
dr.get(“http://www.baidu.com")
dr.implicitly_wait(30)
dr.add_cookie({‘name’:’key—aaaa’,’value’:’value—bbbbb’})
cookies = dr.get_cookies()
for cookie in cookies:
print(“%s ->%s” %(cookie[‘name’],cookie[“value”]))
print(cookies)
dr.quit()

WebDriver原理

Webdriver是按照Server-Client模式设置
Server 端就是Remote Server, 可以是任意浏览器。
Client 端就是测试代码。
工作流程：

  1. WebDriver启动目标浏览器， 并绑定到指定端口。启动的浏览器实例将作为Webdriver 的remote server
  2. Client端通过CommandExcuter发送HTTPRequest给remote server的侦听端口（通信协议：the webdriver wire protocol）
  3. Remote Server 需要依赖原生的浏览器组件（如IEDriverServer.exe、chromedriver.exe）来转化浏览器的native调用。
