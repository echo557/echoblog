find_element_by_xpath()

Absolute Path
/html/body/div/div[3]/form
find_element_by_xpath(“/[@id=’kw’]”)
find_element_by_xpath(“//input[@id=’kw’]”)
find_element_by_xpath(“//[@input=’kw’]”)

find_element_by_xpath(“/html/body/div/div[3]/form”)
find_element_by_xpath(“//*[@input=’kw’ and @id =’wd’]”)
find_element_by_xpath(“//form[@id=’kw’]/span/input”)

其中/表示从根标签开始，//表示不是

Firefox 的插件FirePath可以辅助生成xpath

find_element_by_css_selector()

find_element_by_css_selector(“._cs”)

find_element_by_css_selector(“#id_name”)

#表示id
find_element_by_css_selector(“input”)

#表示tag name

find_element_by_css_selector(“span>input”)

#表示从上一级到下一级
find_element_by_css_selector(“form#form>span>input#kw”)
